Задача №1

1) Создать новый репозитория на GitHub.

2) Создать локально branch "Testing"

3) Создать index.html документ с заголовком HelloWorld, под заголовком любая картинка.

4) Картинка должна находиться в той же папке, что и html док, но в поддиректории /img/

5) Запушить index.html и папку с картинкой на удаленный репозиторий в ветку Testing.

6) Склонировать существующий репозиторий в другую папку.

7) Находясь в директории, в которой был инициализирован репозиторий с самоного начала, создать readme.txt, записать в него какую-либо информацию и запушить на удаленный репозиторий.

8) Перейти в папку с клоном репозитория, и спулить туда изменения. В директории, должен появится тот самый readme.txt.



В ответ на данное упражнение, выслать скриншот, на котором будет видна git bash console иллюстрирующая успешный pull, а также будет отображаться содержимое директории ранее склонированного репозитория и содержимое удаленного репозитория (GitHub).

git bash console (pull)
working directory (cloned repository)
git Hub - remote repository content.
